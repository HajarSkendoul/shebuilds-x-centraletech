# Projects — SheBuilds x CentraleTech

This folder contains all projects submitted for the **SheBuilds x CentraleTech Build Day** (March 8, 2026).

---

## How to submit your project

1. **Fork** this repository
2. Create a folder under `projects/` named after your project:
```
projects/your-project-name/
├── README.md
└── screenshot.png (or demo.gif)
```
3. Your `README.md` must follow this template:

```markdown
## Project Name
**Team:** [Names]
**Demo:** [Live Lovable URL]
**Problem:** What problem does this solve?
**Solution:** How does your app solve it?
**Stack:** Lovable + [anything else]
**Screenshot / Video:** [link or embedded]
```

4. Open a **Pull Request** with the title: `[PROJECT] Your Project Name`
